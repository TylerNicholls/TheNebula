# The Nebula
### A TMT Joint
* Tyler Nicholls
* Michael Spagnolo
* Tim Wilson

Try to knock the guy shooting bullets (enemy) into the nebula (purple circle).
The nebula does damage to the enemy. If you get hit by three bullets, you
die.

Written in rust.
